# 🎯 智能体比赛报名平台 - 快速开始

## 📖 项目简介

这是一个智能体比赛报名网站，支持：
- 🏠 比赛信息展示
- 🔐 CAS单点登录
- 📝 在线报名功能
- 📤 作品提交功能

**新域名**: https://aigctmp.wzbc.edu.cn  
**服务器IP**: 10.145.251.29

---

## ⚠️ 重要提示

**项目目前使用Supabase云数据库**，如果要完全本地部署，需要选择部署方案。

## 🎯 快速部署（3步）

### 第1步：选择方案

阅读 [部署方案说明.md](./部署方案说明.md)，选择：

- **方案一**: 纯展示网站（无数据库，简单）
- **方案二**: 完整功能（需要数据库，复杂）

### 第2步：按照指南部署

打开 [本地部署指南.md](./本地部署指南.md)，按步骤操作：

1. 安装Node.js
2. 构建项目
3. 部署到Apache
4. 配置SSL
5. 测试访问

### 第3步：测试验证

访问 https://aigctmp.wzbc.edu.cn，测试所有功能。

## 📋 部署检查清单

### 环境准备
- [ ] Node.js >= 18 已安装
- [ ] Apache Web服务器已安装
- [ ] SSL证书已准备

### 项目构建
- [ ] 依赖已安装 (`npm install`)
- [ ] 项目已构建 (`npm run build`)
- [ ] dist目录已生成

### 服务器配置
- [ ] 文件已上传到服务器
- [ ] Apache配置已更新
- [ ] SSL证书已配置
- [ ] Apache已重启

### 功能测试
- [ ] HTTPS访问正常
- [ ] 所有页面可访问
- [ ] CAS登录正常
- [ ] CAS注销正常
