# 智能体比赛报名平台 - 使用说明

## 🎯 平台功能

### 普通用户功能
1. **用户注册/登录**
   - 使用用户名和密码注册账号
   - 第一个注册的用户自动成为管理员

2. **在线报名**
   - 填写个人信息（姓名、电话、邮箱、学校等）
   - 填写项目信息（项目名称、描述）
   - 支持团队参赛

3. **作品提交**
   - 上传作品文件（支持图片格式）
   - 自动压缩大于1MB的图片
   - 填写作品链接和描述

4. **报名管理**
   - 查看报名状态（待审核/已通过/未通过）
   - 查看已提交的作品
   - 查看报名详情

5. **比赛规则**
   - 查看参赛要求
   - 查看评分标准
   - 查看奖项设置

### 管理员功能
1. **报名审核**
   - 查看所有报名信息
   - 审核报名（通过/拒绝）
   - 查看参赛作品

2. **用户管理**
   - 查看所有用户
   - 设置用户角色（管理员/普通用户）

## 🚀 快速开始

### 1. 注册账号
- 点击右上角"登录"按钮
- 切换到"注册"标签
- 填写用户名和密码（用户名只能包含字母、数字和下划线）
- 点击"注册"按钮

### 2. 提交报名
- 登录后点击"比赛报名"
- 填写个人信息和项目信息
- 点击"提交报名"
- 等待管理员审核

### 3. 提交作品
- 报名审核通过后，点击"作品提交"
- 选择对应的报名记录
- 填写作品信息
- 上传作品文件或填写作品链接
- 点击"提交作品"

### 4. 查看报名状态
- 点击"我的报名"查看所有报名记录
- 查看审核状态和已提交的作品

## 📝 注意事项

1. **文件上传**
   - 仅支持图片格式（JPEG、PNG、GIF、WEBP）
   - 文件大小限制1MB
   - 超过1MB的图片会自动压缩
   - 文件名不能包含中文字符

2. **报名信息**
   - 报名信息提交后无法修改
   - 请仔细核对后再提交

3. **管理员权限**
   - 第一个注册的用户自动成为管理员
   - 管理员可以在管理后台审核报名和管理用户

4. **重要时间**
   - 报名截止: 2025年12月31日
   - 作品提交: 2026年1月15日
   - 结果公布: 2026年2月1日
